# Introducción a OpenMP — Cálculo Paralelo de la Función Seno

## Descripción General

Este programa implementa el uso de OpenMP para realizar el cálculo paralelo de una sumatoria que involucra la función seno.  
Permite observar el comportamiento y rendimiento de la ejecución cuando se varía el número de hilos de procesamiento.  
También se incluye la medición del tiempo de ejecución en microsegundos para comparar el desempeño entre ejecuciones secuenciales y paralelas.
